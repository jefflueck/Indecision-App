# Indecision-App

## built with React styled using SCSS

---

* This app that will only run on a local machine.

* Built during a Udemy class called **The Complete React Web Developer Course (with Redux)** Instructor: *Andrew Mead*

---
## To run follow these steps.

1. Clone down repository.
2. Install all dependencies.
3. Run *yarn run dev-server* in your terminal.
4. Once compiled go to localhost:8080 on a browser window.  
5. Should be able to use app.
